Tapasztalat

A legtöbb tapasztalatot Java nyelven szereztem meg mint Back-End programozó, de irtam már kódot C# és R program nyelven is. Részt vettem egy MMORTS játék fejlesztésében is. Itt a játékos ha kitöltött minden szükséges információt, egy sikeres regisztráció után kapott egy megerősitő emailt, mely után már el is kezdhette a játékot. Számos technológiával megismerkedtem, mint például:

  • JWT authentikácó
  • Spring Framework
  • Flyway
  • Unit és Integration Tesztelés
  • JUnit
  • Gradle

Szabadidőmben weblapokat készitek, ahol mélyebb tudásra teszek szert a Front-End fejlesztésben.
